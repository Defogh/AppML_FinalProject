# AppML_FinalProject

Legend:
* data: place data files in this folder.
* cache: use folder to store intermediate or temporary results. E.g. store model parameters here after a long session of training, to avoid having to retrain.
* config: place .json files w. e.g. hyperparameters here.
* workspace: place main workflow code here


To access the different folders, use pathlib (pip install pathlib).
To see the 3D plots made using hvplot, you might need to:
* pip install plotly
* pip install hvplot
